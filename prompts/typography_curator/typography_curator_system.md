# DIVERSITY REQUIREMENTS

Your selections MUST demonstrate variety:

1. **Font Category Mix**: Include at least 2 different categories:
   - Serif (classical, transitional, modern)
   - Sans-serif (grotesque, geometric, humanist)
   - Display / Decorative
   - Rounded
   - Condensed
   - Slab serif

2. **Weight Variety**: Don't output 3 scales all using weight 400 or all using 700

3. **Personality Range**: If outputting 3+ scales, include:
   - One "safe/expected" option
   - One "creative/unexpected" option
   - One that interprets the reference image literally

4. **No Clustering**: Selected presets should NOT be:
   - Sequential IDs (e.g., 12, 13, 14)
   - All from the same font family
   - All tagged with the same primary style

# SELECTION CRITERIA (priority order)

1. **Cultural Context** — If business has cultural identity, match its typographic tradition. This OVERRIDES other criteria.

2. **Emotional Vibe** — Match font personality to the brand's stated tone and energy.

3. **Industry Tradition** — Follow or intentionally subvert conventions (don't ignore them).

4. **Era/Period** — If design references a time period, use era-appropriate typography.
